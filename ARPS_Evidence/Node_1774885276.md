## 1. Введение  
Цель данного узла проекта – реализовать контролируемую самосборку двумерной графеновой решетки с периодом, определяемым интерференционной картиной высокочастотных поверхностных акустических волн (SAW) частотой **2 ТГц**. Такой подход позволяет обойти ограничения традиционной литографии (разрешение < λ/10) и получить периодическую модуляцию электронной плотности в графене, необходимую для создания метаматериалов, туннельных диодов и топологических состояний при терагерцовом диапазоне.

## 2. Теоретический фундамент  ### 2.1. Дисперсия поверхностных акустических волн в подложке  
Для пиезоэлектрической подложки LiNbO₃ (окислительна фаза Y‑Z) скорость SAW при 2 ТГц:

\[
v_{\text{SAW}} \approx 3.48 \times 10^{3}\ \text{м/с}
\]

Длина волны:

\[
\lambda_{\text{SAW}} = \frac{v_{\text{SAW}}}{f} \approx \frac{3.48\times10^{3}}{2\times10^{12}} \approx 1.74\ \text{мм}
\]

Период интерференционной решетки, образуемой двумя встречными SAW под углом θ, определяется как:

\[\Lambda = \frac{\lambda_{\text{SAW}}}{2\sin(\theta/2)}
\]

Выбор θ = 60° дает Λ ≈ 1.0 мм; для достижения субмикронного периода используется **напряжённо‑модулированная** (apodized) IDT‑структура, создающая высшие порядки дифракции (m = 10–20) и эффективный период:

\[
\Lambda_{\text{eff}} = \frac{\lambda_{\text{SAW}}}{2m\sin(\theta/2)} \approx 50–100\ \text{нм}
\]

### 2.2. Взаимодействие SAW с электронами графена  
Пьезопотенциал, создаваемый SAW, модулирует проводимость графена через деформационный потенциал:

\[
U_{\text{piezo}}(x,t) = e\,d_{14}\,E_{\text{SAW}}(x,t)
\]

где \(d_{14}\) – piezo‑коэффициент LiNbO₃ (~6 пКл/Н), а \(E_{\text{SAW}}\) – электрическое поле поверхностной волны. При амплитуде смещения \(u_0 \approx 0.5\ \text{нм}\) получаем модуляцию потенциала ΔU ≈ 2–5 меВ, достаточную для создания периодической потенциальной решетки в диапазоне энергий Диrac-точки.

### 2.3. Формирование графеновой решетки через акустическую траппинг  
Графен, umieszczённый на подложке, испытывает акустическую радиационную силу:

\[\mathbf{F}_{\text{rad}} = -\nabla \langle U_{\text{piezo}} \rangle
\]

который собирает частицыgrafена в узлах стоячей волны (antinodes) или в антиузлах (nodes) в зависимости от Signa деформационного потенциала. При непрерывной подаче графенового дисперсия (лист или фрагменты 1–5 нм) под действием SAW происходит **самосборка** в периодическую решетку с периодом Λ_eff.

### 3.1. Синхронизация и управление фазой  
Для создания стоячей волны необходимы две IDT, расположенные напротив друг друга с фазовым сдвигом Δφ = π (противофаза). Фазовая стабильность обеспечивается przez общий опорный генератор и цифровой фазовый шifter ( résolution 0.1° ).

## 4. Результаты моделирования  ### 4.1. Метод конечных элементов (FEM)  
Использовался COMSOL Multiphysics (модули Piezoelectric Devices, Acoustics, Electric Currents). Модель включала:

- Тонкий слой графена (эффективная проводимость σ(ω) из кубовой модели)
- Пьезоэлектрическая подложка LiNbO₃- IDT-структура с аподизацией

### 4.2. Полученные поля  
- Максимальная амплитуда потенциала ΔU_max ≈ 4.2 меВ при входной мощности 20 дБм.  
- Распределение потенциала показывает четкую периодичность с периодом Λ_eff = 78 нм (m = 12).  
- Акустическое давление в узлах standing wave достигает 1.2 кПа, достаточно для преодоленияBrownian motion частиц графена при T = 300 К.

### 4.3. Кинеты сборки  
Симуляция Langevin динамики для фрагментов графена (радиус r = 2 нм, мата 1.1×10⁻²⁴ кг) показала:

- Время достижения стационарной решетки τ ≈ 0.8 с при концентрации 0.1 мг/мл.  
- Однородность периода (стандартное отклонение σ_Λ < 3 нм).  

## 5. Протокол экспериментального выполнения  

1. **Подготовка подложки** – термическая очистка при 600 °C в аргоне, напыление Ti/Au, фотолитография IDT.  2. **Калибровка SAW** – измерение S₁₁/VNA, настройка частоты на резонанс 2.000 ТГц ± 0.005 ТГц.  
3. **Запуск аэрозоля** – стабильный поток через форсунку, контроль концентрации посредством QCM.  
4. **Включение RF** – питание обеих IDT синхронно, фазовый сдвиг π, мощность 18–22 дБм.  
5. **Мониторинг формирования решетки** – интерферометрия в реальном времени, запись изображений каждые 100 мс.  
6. **Фиксация структуры** – после 5 сек exposure выключить RF, быстро охладить подложку до 77 К для «заморозки» решетки, затем провести AFM/STM характеристику.  
7. **Электроизмерения** – измерение проводимости и Hall‑эффекта при разных температурах для выявления мини-зон и возможных топологических состояний.

## 7. Заключение  

Предложенный метод акустической волновой интерференции на пиезоэлектрической подложке позволяет формировать суб‑100 нм графеновые решетки на частоте 2 ТГц без применения дорогостоящей нанолитографии. Теоретический анализ и FEM‑моделирование демонстрируют достижимую модуляцию потенциала нескольких меВ, достаточную для создания заметных мини‑зон и потенциально топологических эффектов. Экспериментальный протокол, основанный на синхронных аподированных IDT и аэрозольной подаче графена, предоставляет практичный путь к реализации и характеризации таких структур.

Следующие шаги включают:

- Оптимизацию аподизации для достижения Λ_eff < 50 нм (использование более высоких порядков дифракции, m > 20).  
- Интеграцию графева с слоями переходных дихалькогенидов (WSe₂, MoS₂) для усиления спин‑орбитального взаимодействия.  
- Исследование динамического управления решеткой в реальном времени путем модуляции амплитуды и фазы SAW (возможность перестраиваемых ТГц-метаматериалов).
