# Metal hydride material requirements for automotive hydrogen storage systems

**🧬 HASH:** `79fa856ceec304e2b1f47d91cc477ee2c4d29caa6a94cd5c9f1304777002d3d9`

**Техническийотчёт (краткий)**  
**Тема:** *Metal‑hydride material requirements for automotive hydrogen‑storage systems*  
**Источник:** цели DOE (2017) для бортовых перезаряжаемых систем хранения H₂ в лёгковых автомобилях  

---

### 1. Цели DOE (2017) для на‑бордных систем  
| Показатель | Целевое значение* |
|------------|-------------------|
| Гравелиметрическая ёмкость | ≥ 5,5 мас.% H₂ |
| Объёмная ёмкость | ≥ 40 г H₂/л |
| Рабочее давление | 1–10 бар (наполнение/разряд) |
| Диапазон температур работы | –40 °C … +60 °C |
| Время заправки (90 % ёмкости) | ≤ 3 мин |
| Циклическая стойкость | ≥ 1000 полных циклов (≈ 1500 ц. с запасом) |
| Кратковременная мощность (разряд/поглощение) | ≥ 1,5 кВт·кг⁻¹ (эквив. ≥ 0,5 Гц) |
| Энергетическая эффективность (round‑trip) | ≥ 80 % |

\*Значения указаны для чистой системы; при наличии баланс‑оф‑плант (BoP) компонентов требуются поправки.

---

### 2. Методология оценки материальных требований  
1. **Упрощённая модель системы** – учитываются только активный гидрид, оболочка и минимальные BoP элементы (теплообменник, клапаны, датчики). Предполагается, что BoP добавляет ~10 % к массе и ~15 % к объёму системы.  
2. **Интерполяция данных из баз металл‑гидридов** (например, Sandia Hydride Database, NIST) – подбор сплавов, удовлетворяющих интерполированным точкам по ёмкости, давлению равновесия и температуре разложения.  
3. **Расчёт требуемых показателей материала** – к целевым значениям системы добавляются поправки BoP и запас по kinetics (≈ 20 % запас по скорости поглощения/выделения).  

---

### 3. Полученные требования к металл‑гидридам (с учётом Ni/Co‑легированных систем)

| Параметр | Требуемое значение (материал) | Пояснение |
|----------|------------------------------|-----------|
| **Гравелиметрическая ёмкость** | **≥ 6,2 мас.% H₂** | С учётом 10 % BoP‑массы → система достигает 5,5 мас.% |
| **Объёмная ёмкость** | **≥ 45 г H₂/л** | 15 % BoP‑объёма → система ≥ 40 г/л |
| **Давление равновесия (P_eq)** | 0,5–5 бар при T = –20 °C … +80 °C | Позволяет работать в диапазоне DOE без значительного нагрева/охлаждения |
| **Температура разложения (T_des)** | 60–100 °C (при P = 1 бар) | Достаточно низкая для использования отработанного тепла двигателя/выхлопа |
| **Температура поглощения (T_abs)** | ≤ 30 °C (при P = 5 бар) | Обеспечивает быструю заправку при ambient‑температуре |
| **Кинетика (t_90%)** | ≤ 30 с для 90 % поглощения/выделения при ΔP = 5 бар | Соответствует частоте ≥ 0,5 Гц (30 циклов/мин) – достаточно для заправки за ≤ 3 мин |
| **Циклическая стойкость** | ≥ 1500 полных циклов (≤ 0,1 % деградации ёмкости за цикл) | С запасом к цели DOE 1000 циклов |
| **Толерантность к примесям** | ≤ 10 ppm CO, ≤ 5 ppm S, ≤ 10 ppm H₂O без заметной потери ёмкости | Соответствует обычному качеству водорода из реформинг‑установок |
| **Энергетическая эффективность (round‑trip)** | ≥ 82 % (энергия выделенного H₂ / энергия, затраченная на поглощение) | Достигается при минимальных тепловых потерях (ΔT < 15 °C) и высокой обратимости гидрид‑дегидридной реакции |
| **Состав (Ni/Co‑легированные)** | Примеры: **LaNi₅‑xCoₓ** (x = 0,2–0,5), **Mg₂Ni₁‑yCo_y** (y = 0,1–0,3), **TiFe₀,₈Ni₀,₂** | Добавление Co повышает гидридообразующую способность, снижает T_des и улучшает циклическую стабильность; Ni обеспечивает высокую intrinsic ёмкость и хорошую проводимость тепла. |

### 4. Выводы  

1. Для достижения целевых показателей DOE 2017 в бортовых системах хранения водорода необходимо, чтобы металл‑гидрид обладал гравелиметрической ёмкостью **≥ 6,2 мас.%** и объёмной ёмкостью **≥ 45 г H₂/л** (с учётом минимального BoP).  
2. Сплавы на основе **Ni/Co** (LaNi₅‑xCoₓ, Mg₂Ni‑yCo_y, TiFe‑Ni) удовлетворяют требуемому диапазону давлений равновесия (0,5–5 бар) и температур работы (–20 °C … +80 °C) при минимальном тепловом управлении.  
3. Кинетика поглощения/выделения должна обеспечивать **t₉₀% ≤ 30 с**, что соответствует рабочей частоте **≥ 0,5 Гц** (≈ 30 циклов в минуту) и позволяет выполнить заправку за ≤ 3 мин.  
4. Ожидаемая **энергетическая эффективность** системы на базе таких гидридов составляет **≥ 80 %**, а циклическая стойкость не менее **1500 полных циклов** гарантирует соответствие цели DOE по долговечности.  
5. При использовании минимальных BoP элементов (теплообменник, клапаны, датчики) материальные требования, приведённые выше, являются достаточными для построения коммерчески жизнеспособной автомобильной водородной системы, удовлетворяющей всем целевым показателям DOE 2017.

*Примечание:* Приведённые числа являются расчётными оценками на основе упрощённой модели и интерполированных данных из открытых баз металл‑гидридов. Для окончательного выбора конкретного сплава необходимы экспериментальные проверки в условиях реального автомобильного цикла (термические циклы, загрязнения, вибрационные нагрузки).
