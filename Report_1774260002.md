# General dispersion properties of magnetized plasmas with drifting bi-Kappa distributions. DIS-K: DIspersion Solver for Kappa plasmas

**Технический отчёт**  
*Статья:* *General dispersion properties of magnetized plasmas with drifting bi‑Kappa distributions. DIS‑K: DIspersion Solver for Kappa plasmas*  

---

### 1. Цель работы  
Изучить влияние не‑тепловых, анизотропных и дрейфующих распределений скоростей на дисперсионные свойства волн в магнитосредней плазме, а также представить числовой инструмент **DIS‑K** для быстрого расчёта таких дисперсий.

### 2. Физический контекст  
- Наблюдения в солнечном ветре, магнитосфере и астрофизических потоках показывают отклонения от максвелловского распределения: наличие анизотропий относительно направления магнитного поля **B₀**, полевых пучков (field‑aligned beams) и надтепловых «хвостов».  - Такие особенности удобно описываются **дрейфующей би‑Каппа‑функцией**  

\[
f_{\kappa}(\mathbf{v})=
\frac{n}{(\pi\kappa\theta_{\perp}^{2})^{3/2}}
\frac{\Gamma(\kappa+1)}{\kappa^{3/2}\Gamma(\kappa-1/2)}
\left[1+\frac{1}{\kappa}\left(
\frac{v_{\perp}^{2}}{\theta_{\perp}^{2}}+
\frac{(v_{\parallel}-U_{d})^{2}}{\theta_{\parallel}^{2}}
\right)\right]^{-\kappa-1},
\]

где  - \(n\) – плотность частиц,  
- \(\theta_{\perp,\parallel}=\sqrt{2k_{B}T_{\perp,\parallel}/m}\) – тепловые скорости,  
- \(U_{d}\) – дрейф вдоль **B₀**,  
- \(\kappa>3/2\) управляет степенью надтеплости (чем меньше κ, тем более «тяжёлый» хвост).  

При \(\kappa\rightarrow\infty\) функция переходит к анизотропному Максвеллу.

### 3. Дисперсионное уравнение  Для волн с волновым вектором **k** и частотой \(\omega\) в холодной магнитосредней плазме с учётом би‑Каппа распределения получаем общее дисперсионное уравнение в виде тензорного отношения:

\[
\mathbf{D}(\omega,\mathbf{k})\cdot\mathbf{E}=0,
\qquad 
\mathbf{D}= \mathbf{I}c^{2}\frac{k^{2}}{\omega^{2}}-
\sum_{s}\frac{\omega_{ps}^{2}}{\omega^{2}}
\int\! \frac{\mathbf{v}\mathbf{v}}{v^{2}}
\frac{\partial f_{\kappa,s}}{\partial \mathbf{v}}\,d^{3}v\Bigg/ \left(\omega-\mathbf{k}\!\cdot\!\mathbf{v}\right),
\]

где суммирование происходит по всем видами частиц \(s\), \(\omega_{ps}\) – плазменная частота. Интеграл сводится к специальным функциям (Гамма, гипергеометрическая \(_2F_1\)) и может быть вычислен аналитически для произвольного угла между **k** и **B₀**.

### 4. Основные результаты  
- **Анизотропия температуры** (\(T_{\perp}\neq T_{\parallel}\)) приводит к смещению и расширению полос нестабильности (например, огненно‑ионосферные и альфvén‑ионосферные нестабильности) по сравнению с Максвелловым случаем.  
- **Дрейф \(U_{d}\)** вдоль поля смещает резонанс \(\omega-\mathbf{k}_{\parallel}U_{d}=0\), усиливая нестабильность поток‑две‑потока и позволяя возбуждать ветровые волны при меньших скоростях дрейфа.  
- **Надтепловой хвост** (низкий κ) увеличивает вклад высокоэнергетических частиц в дисперсионную функцию, что приводит к:  
  * уменьшению пороговой скорости дрейфа для возникновения нестабильности,  
  * усилению демпфирования (ландовега) высокочастотных ветровых волн,  
  * появлению дополнительных ветвей, связанных с резонансом на высоких гармониках циклotron‑частоты.  
- При κ ≈ 2–4 (значения типичные для солнечного ветра) эффекты наиболее выражены; при κ > 10 результаты сходится к классическим анизотропным Максвелловым моделям.

### 5. Описание пакета **DIS‑K**  
- **Язык:** Python 3 (сопровождается модулем на Cython для критических циклов).  
- **Функциональность:**  
  * Ввод параметров плазмы (плотности, температуры, анизотропии, дрейфы, κ).  
  * Выбор типа волны (Электростатическая, электромагнитная, альфvén, магнитозвуковая).  
  * Автоматическое построение дисперсионного отношения \(\det\mathbf{D}=0\) и поиск корней \(\omega(k)\) в комплексной плоскости (метод Ньютона‑Рафсона с продолжением по параметрам).  
  * Вывод: диаграммы дисперсии, коэффициенты роста/затухания, карты нестабильности в пространстве \((k_{\parallel},k_{\perp})\).  
- **Валидация:** Сравнение с published результатами для би‑Мах distribución (κ→∞) и с опубликованными данными по наблюдениям в магнитосфере MMS. Отклонения < 2 % для диапазона κ = 2–10.  
- **Доступность:** Репозиторий на GitHub (лицензия MIT), включает примеры Jupyter‑ноутбуков и документацию (Sphinx).

### 6. Выводы  
- Дрейфующая би‑Каппа функция предоставляет компактное, физически обоснованное описание пространственных плазм, Simultaneously учитывая анизотропию, дрейф и надтепловые хвосты.  
- Аналитическое исследование дисперсионного отношения показывает, что все три эффекта существенно модифицируют положения порогов и темпов роста классических плазменных нестабильностей.  
- Пакет **DIS‑K** позволяет быстро исследовать эти модификации в широком диапазоне параметров, что делает его полезным инструментом для интерпретации спутниковых измерений и для планирования лабораторных экспериментов с магнитосредней плазмой.
