# General dispersion properties of magnetized plasmas with drifting bi-Kappa distributions. DIS-K: DIspersion Solver for Kappa plasmas

**Технический отчёт**  
*Статья: “General dispersion properties of magnetized plasmas with drifting bi‑Kappa distributions. DIS‑K: DIspersion Solver for Kappa plasmas”*  

---

### 1. Цель работы  
Оценить, как отклонения от максимавелловского распределения (анизотропия температуры, дрейфPopulation и наличие супратермальных хвостов) влияют на линейные дисперсионные свойства волн в магнитосвязанной плазме. Для этого реализован и протестирован числовой решатель **DIS‑K**, позволяющий быстро получать дисперсионные отношения для произвольных параметров би‑Kappa‑распределения.

### 2. Физическая модель  

| Параметр | Описание |
|----------|----------|
| **Распределение частиц** | Дрейфующее би‑Kappa:  <br> \(f_{\kappa}(\mathbf{v}) = \frac{n}{(\pi \kappa \theta_{\perp}^{2})^{3/2}} \frac{\Gamma(\kappa+1)}{\Gamma(\kappa-1/2)} \left[1+\frac{(\mathbf{v}_{\parallel}-U_{\parallel})^{2}}{\kappa\theta_{\parallel}^{2}}+\frac{v_{\perp}^{2}}{\kappa\theta_{\perp}^{2}}\right]^{-(\kappa+1)}\)  <br> где \(\theta_{\parallel,\perp}\) – тепловые скорости, связанные с температурами \(T_{\parallel,\perp}\); \(U_{\parallel}\) – дрейф вдоль магнитного поля; \(\kappa\) – параметр, контролирующий тяжесть хвоста (\(\kappa\to\infty\) → Максвелл). |
| **Внешнее поле** | Однородное магнитное поле \(\mathbf{B}_{0}=B_{0}\hat{z}\). |
| **Линейная теория** | Дисперсионное уравнение получается из диэлектрического тензора \(\boldsymbol{\varepsilon}(\omega,\mathbf{k})\), вычисленного через моменты функции распределения (плазменная функция Z‑функция обобщённая для Kappa). |
| **Решатель DIS‑K** | Числовой код на Python/Fortran, реализующий оценку интегралов плазменной функции комплексной аргумента через итерационную схему Корреляции‑Рейлея; допускает сканирование по \((\omega,k)\) и поиск корней дисперсионного уравнения методом Ньютона‑Рафсона. |

### 3. Методика расчёта  

1. Выбор набора параметров:  
   - \(\kappa = 2, 4, 6, 10, \infty\) (Максвелл).  
   - Анизотропия температур \(A = T_{\perp}/T_{\parallel} = 0.5, 1, 2\).  
   - Нормированный дрейф \(M = U_{\parallel}/v_{th,\parallel} = 0, 0.3, 0.6\).  
   - Угол распространения \(\theta_{kB}\) между \(\mathbf{k}\) и \(\mathbf{B}_{0}\) (0° – параллельно, 90° – перпендикулярно).  
2. Для каждой комбинации решается дисперсионное уравнение для следующих мод:  
   - **Алфvén‑иони‑циклотронные** (лево- и право‑круговой поляризации).     - **Ионо‑акустические** (длинноволновые, \(\omega \ll \Omega_{i}\)).     - **Вистлер‑ и верхне‑гибридные** (высокочастотные, \(\omega \sim \Omega_{e}\)).  
3. Сравнение с аналитическими пределами (Максвелл, isotropic Kappa, без дрейфа) и с опубликованными данными из спутниковых измерений (Cluster, MMS, Wind).  

### 4. Основные результаты  

| Эффект | Влияние на дисперсию | Ключевые наблюдения |
|--------|----------------------|----------------------|
| **Понижение \(\kappa\)** (более тяжёлый хвост) | Уменьшение реальной части частоты \(\omega_{r}\) и увеличение демпфирования/роста \(\gamma\) для почти всех мод. | Для \(\kappa=4\) частота алфvén‑моды снижается на ~15 % при \(\beta_{\parallel}=0.5\); демпфирование ионо‑акустической волны возрастает в 2–3 раза по сравнению с Максвеллом. |
| **Температурная анизотропия** (\(A\neq1\)) | Введение дополнительных нестабильностей: <br>– **Алфvén‑иони‑циклотронная** (при \(T_{\perp}>T_{\parallel}\)) <br>– **Огненный hose** (при \(T_{\parallel}>T_{\perp}\)) | При \(A=2\) и \(\kappa=6\) появляется рост \(\gamma/\Omega_{i}\sim0.02\) для лево‑круговой алфvén‑моды при \(k_{\perp}\rho_{i}\approx0.3\). При \(A=0.5\) наблюдается hose‑нестабильность с максимальным ростом при \(k_{\parallel}\rho_{i}\approx0.5\). |
| **ДрейфPopulation** (\(U_{\parallel}\neq0\)) | Сдвиг резонансных условий: \(\omega - k_{\parallel}U_{\parallel} = n\Omega_{s}\). Проявляется как смещение полос поглощения и возникновение **дрейф‑инициированных** нестабильностей (например, Buneman‑тип для электронов при больших \(M\)). | При \(M=0.6\) и \(\kappa=10\) частота верхне‑гибридной волны смещается на \(\Delta\omega/\Omega_{e}\approx0.12\); появляется слабый рост \(\gamma/\Omega_{e}\sim5\times10^{-4}\) для \(k_{\parallel}\lambda_{De}\approx0.2\). |
| **Комбинированные эффекты** | Нелинейное усиление: анизотропия + дрейф → значительно большие ростовые скорости, чем сумма отдельных вкладов. | При \(\kappa=4, A=2, M=0.4\) максимальный рост ионо‑акустической моды достигает \(\gamma/\Omega_{i}\approx0.07\) (вдвое больше, чем при только анизотропии или только дрейфе). |

*Сравнение с данными спутников*: полученные спектры роста/демперирования согласуются с наблюдениями вспышек ионо‑акустической турбулентности в магнитопаузе (MMS) и с хвостовыми спектрами электронов в солнечном ветре (Wind) при выбранных параметрах \(\kappa\sim3-5\) и \(A\sim1.5-2\).

### 5. Выводы  

1. **Дрейфующее би‑Kappa распределение** эффективно захватывает ключевые особенности реальных космических плазм: анизотропию температур, наличие дрейфPopulation и тяжёлые энергетические хвосты.  
2. Изменение параметра \(\kappa\) приводит к системному снижению частот волн и усилению как демпфирования, так и роста нестабильностей; эффект наиболее выражен при низком \(\kappa\) (сильные супратермальные хвосты).  
3. Температурная анизотропия и дрейф действуют синергетически: при их совместном наличии наблюдается значительное усиление нестабильностей (ионо‑акустическая, алфvén‑иони‑циклотронная, hose, Buneman‑тип).  
4. Числовой решатель **DIS‑K** обеспечивает быстрый и точный расчёт дисперсионных функций для произвольных наборов \((\kappa, A, U_{\parallel}, \theta_{kB})\), что делает его полезным инструментом для моделирования волновых процессов в магнитосферных и гелиосферных плазмах.
