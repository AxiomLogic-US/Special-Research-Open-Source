# Entangling Disciplines: Causality, Entropy and Time-Travel Paradoxes on a Quantum Computer

**Технический отчет: Интеграция квантовых вычислений с фундаментальной физикой для исследования причинности, энтропии и парадоксов времени**  
**Цель:** Показать, как квантовые вычисления могут служить экспериментальной платформой для изучения открытых проблем фундаментальной физики (причинность, термодинамика, парадоксы времени travel), одновременно обогащая обучение квантовой информатике (КИ) через практически реализуемые эксперименты на современном оборудовании.  **Методология:**  
В работе предложено выполнение серии квантовых схем на текущих NISQ-процессорах (Noisy Intermediate-Scale Quantum), моделирующих ключевые физические сценарии:  
- **Причинность и закрытыевременные контуры (ЗВК):** Реализация моделей типа Deutsch для самосогласованных решений paradoxов (например, "дедушкин парадокс") посредством квантовых операций с пост-селекцией и ancilla-кубитами, позволяющих проверять условия самосогласованности без нарушения линейности квантовой механики.  
- **Энтропия в причинных петлях:** Измерение квантовой энтропии (фон Неймана) в схемах, имитирующих обратное время эволюции, для анализа соответствия второму закону термодинамики в нестандартных причинных структурах.  - **Практическая реализация:** Использование базовых гейтов (Hadamard, CNOT, измерения) и техник смягчения шума (например, нулевая экстраполяция) для преодоления ограничений текущего оборудования при малом числе кубитов (<10).  

**Ключевые результаты:**  
- Экспериментально подтверждена возможность получения самосогласованных решений для ЗВК-схем на реальных квантовых процессорах (например, IBM Quantum, Rigetti), демонстрирующих разрешение парадоксов через когерентное суперпозиционное состояние вместо классической логики.  
- Изменения энтропии в смоделированных причинных петлях показали соответствие ожиданиям из квантовой теории информации (например, снижение энтропии в post-seleцированных подансамблях компенсируется ростом в отбрасываемых частях), поддерживая согласованность с generalized second law.  
- Схемы успешно выполнялись с фidelностью >70% после коррекции ошибок на 5–7 кубитах, подтверждая практическую осуществимость предложенного подхода для образовательных и исследовательских целей.  

**Значение и выводы:**  
Данный подход иллюстрирует синергетический эффект междисциплинарной интеграции:  
1. Для физики — предоставляет доступный способ тестирования спекулятивных моделей (квантовая гравитация, квантовая термодинамика) без потребности в экзотических физических системах.  2. Для КИ — создает мотивирующие учебные лаборатории, где абстрактные концепции (запутанность, измерение, коррекция ошибок) осваиваются через решение tangible физических задач.  
3. Для образования — формирует навыки перехода между дисциплинами, критически важные для прогресса в квантовых технологиях.  
**Заключение:** Работа демонстрирует, что современные квантовые компьютеры уже могут выступать не только как вычислительные устройства, но и как экспериментальные столы для фундаментальной науки, превращая философские парадоксы в проверяемые инженерные задачи при сохранении строгой научной методологии.
